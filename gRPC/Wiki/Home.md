## ðŸš€ About This Project
gRPC is a modern, open-source, high-performance RPC framework that can run in any environment. It uses **Protocol Buffers (Protobuf)** as its Interface Definition Language (IDL).

### Key Features Covered:
*   **Protobuf (.proto)**: Defining messages and services.
*   **Services**: implementing business logic.
*   **Streaming**: Server-side, Client-side, and Bi-directional streaming.
*   **Interceptors**: Middleware-like logic for logging, validation, and auth.
*   **Performance**: Why gRPC is faster than REST.
